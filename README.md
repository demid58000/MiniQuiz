## Мини‑викторина (HTML + CSS + JS)

Небольшой учебный проект на чистом JavaScript: одностраничная **мини‑викторина** с вопросами по основам JS, подсчётом очков и экраном результатов.

### Как запустить

- **Вариант 1 (простой)**:  
  Открой файл `index.html` двойным кликом в браузере.

- **Вариант 2 (через Live Server / локальный сервер)**:  
  Открой папку `mini-quiz` в редакторе (VS Code / Cursor) и запусти `index.html` через расширение **Live Server** (или любой другой локальный сервер), чтобы удобнее разрабатывать.

### Файлы проекта

- **`index.html`** — разметка страницы, три экрана:
  - стартовый экран с кнопкой «Начать»;
  - экран с вопросами и вариантами ответов;
  - экран с итоговым результатом и кнопкой «Пройти ещё раз».
- **`style.css`** — оформление:
  - тёмная тема, карточка с лёгкой тенью;
  - стили для кнопок, ответов, подсветка правильных/неправильных вариантов;
  - адаптивная верстка под небольшие экраны.
- **`script.js`** — логика:
  - массив вопросов `questions` с вариантами ответов и индексом правильного;
  - переход между экранами (старт → викторина → результат);
  - обработка клика по ответу, подсветка правильного/неправильного;
  - подсчёт очков и вывод процента правильных ответов.

### Как устроен JavaScript‑код

1. **Данные**  
   В начале файла объявлен массив `questions`. Каждый объект — один вопрос:
   - `text` — текст вопроса;
   - `answers` — массив строк с вариантами;
   - `correctIndex` — индекс правильного варианта.

2. **Состояние**  
   Переменные:
   - `currentQuestionIndex` — номер текущего вопроса;
   - `score` — количество правильных ответов;
   - `answered` — флаг, отвечал ли пользователь на текущий вопрос.

3. **Основные функции**
   - `showQuestion()` — показывает текущий вопрос и создаёт кнопки‑ответы;
   - `selectAnswer(button, index)` — вызывается при клике по варианту:
     - отмечает правильный и (если нужно) неправильный ответ;
     - обновляет счёт, включает кнопку «Следующий вопрос»;
   - `showResult()` — показывает экран результата и текст с процентом;
   - `switchScreen(from, to)` — переключает видимые блоки;
   - `resetQuiz()` — сброс состояния для нового прохождения.

4. **События**
   - `startBtn` — запускает викторину с первого вопроса;
   - `nextBtn` — переходит к следующему вопросу или показывает результат;
   - `restartBtn` — сбрасывает викторину и начинает заново.

### Идеи для доработки

- **Добавить больше вопросов** или сделать свою тему (HTML, CSS, Python и т.д.).
- **Таймер** на вопрос или на всю викторину.
- **Сохранение лучшего результата** в `localStorage`.
- **Экран настроек** (выбор количества вопросов, уровень сложности).
